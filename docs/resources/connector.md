# customer-connect_connector (Resource)

Manages a NetFoundry Connector.

## Example Usage

```terraform
resource "customer_connect_location" "example" {
  customer_id = "00000000-0000-0000-0000-000000000001"
  name        = "NYC Office"
}

# GATEWAY connector attached to a location
resource "customer_connect_connector" "gateway" {
  location_id = customer_connect_location.example.id
  name        = "nyc-gateway-01"
  description = "Primary gateway connector for NYC Office"
  type        = "GATEWAY"
}

# DEVICE connector attached to a location
resource "customer_connect_connector" "device" {
  location_id = customer_connect_location.example.id
  name        = "nyc-device-01"
  description = "Device connector for NYC Office"
  type        = "DEVICE"
}

# Connector created from a connector model (type is derived from the model)
resource "customer_connect_connector" "from_model" {
  location_id        = customer_connect_location.example.id
  name               = "nyc-model-connector-01"
  connector_model_id = "00000000-0000-0000-0000-000000000002"
}
```

## Schema

### Required

- `location_id` (String) Location this connector belongs to. Changing this forces a new resource.
- `name` (String) Name of the connector.

### Optional

- `connector_model_id` (String) Connector model to inherit applications from.
- `description` (String) Description of the connector.
- `enabled` (Boolean) Whether the connector is enabled. Defaults to true on creation.
- `type` (String) Type of the connector: `DEVICE`, `GATEWAY`, or `SDK_EMBEDDED`. Required unless `connector_model_id` is set, in which case the type is derived from the model.

### Read-Only

- `created_at` (String) Timestamp when this connector was created.
- `created_by` (String) Identity that created this connector.
- `customer_id` (String) Customer this connector belongs to. Omitted when the connector is hosted directly by the provider.
- `deleted` (Boolean) Whether this connector has been deleted.
- `deleted_at` (String) Timestamp when this connector was deleted.
- `deleted_by` (String) Identity that deleted this connector.
- `enrolled` (Boolean) Whether the connector is enrolled in Ziti.
- `enrollment_expires_at` (String) Expiration timestamp of the current Ziti enrollment.
- `enrollment_jwt` (String, Sensitive) Enrollment JWT for the connector, present until the connector enrolls.
- `id` (String) Unique identifier of the connector.
- `online` (Boolean) Whether the connector is online in Ziti.
- `owner_identity_id` (String) Identity that owns this connector.
- `provider_id` (String) Provider this connector belongs to.
- `updated_at` (String) Timestamp when this connector was last updated.
- `ziti_id` (String) ID of the Ziti entity backing this connector.
- `ziti_name` (String) Unique name used for the Ziti identity.
